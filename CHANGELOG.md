# 更新日志

本文档记录项目的所有重要更改。

## [3.0.0] - 2025-02-15 (最新版本)

### ✨ 新增
- 多机器人协同控制功能
  - 多机器人任务分配
  - 集群导航控制
  - 碰撞避免策略
- 高级导航功能
  - 动态路径规划
  - 自适应速度控制
  - 多目标点导航
- 3D可视化功能
  - 3D地图显示
  - 机器人姿态可视化
  - 传感器数据3D显示

### 🔄 改进
- 优化多机器人协同算法
- 改进3D渲染性能
- 增强网络通信稳定性

### 🐛 修复
- 修复多机器人通信延迟问题
- 解决3D显示卡顿问题
- 修复内存泄漏问题

### 🔄 改进

## [1.0.0] - 2025-2-5 (首个稳定版本)

### ✨ 新增
- 完整的 ROS 网络配置界面
  - 主从机模式切换
  - 自动检测本地网络接口
  - 连接状态可视化
- 多种 SLAM 建图方法支持
  - Gmapping SLAM
  - Cartographer SLAM
  - Hector SLAM
- 虚拟摇杆遥控功能
  - 线速度和角速度控制
  - 碰撞避免系统
- 智能导航系统
  - 多种路径规划算法
  - 自适应定位
  - 动态避障
- 参数配置界面
  - 机器人参数配置
  - 导航参数调整
  - 配置文件导入导出

### 🔄 改进
- 优化了网络连接的稳定性
  - 自动重连机制
  - 连接状态实时监控
- 改进了建图过程的实时显示
  - 地图更新更流畅
  - 添加进度显示
- 增强了导航的安全性
  - 完善的碰撞检测
  - 动态避障优化
- 优化了用户界面布局
  - 更直观的控制面板
  - 更清晰的状态显示

### 🐛 修复
- 修复了网络断开时的崩溃问题
- 修复了地图保存可能失败的问题
- 修复了某些参数设置无效的问题
- 解决了多个内存泄漏问题
